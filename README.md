
== CIFS@Etersoft ==

This package contains Etersoft modified CIFS kernel module
with WINE@Etersoft sharing access support.

etercifs is a patched cifs module with wine file locking support (NT semantic)


FILE_SHARE_READ
FILE_SHARE_WRITE
FILE_SHARE_DELETE
